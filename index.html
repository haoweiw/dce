Welcome to the DHCPD Configuration Editor.

DHCPD Configuration Editor is a web service to update MAC address in a dhcpd.conf. It allows to:

  • load a dhcpd.conf file from a dhcp-server folder,
  • save a modified dhcpd.conf file,
  • apply modification on server (restart DHCPD daemon, this function is disabled to prevent faulty activity),
  • import a dhcpd.conf file from local file-system,
  • export a modified dhcpd.conf to local file-system,
  • and, obviously, change MAC address of previously declared host entries.

The user interface is split into 5 area. On top and in the bottom, there are descriptive header and footer. On left there a menu panel follow by a short sub area displaying useful tricks. The central area shows main information and control button. Just between this area and the footer, there's a status bar to display informative messages and errors.

Introduction

Left side menu bar contains 6 buttons. The first one, named "Home", is the tab containing current general information of this web service. The second button, named "Edit", switches to the editor tab where entry forms are present to update MAC addresses. Other buttons are linked to direct actions such as.

  • the "Load" button which is used to load a dhcpd.conf file from dhcp-server folder,
  • the "Save" button which is used to save a modified dhcpd.conf file,
  • the "Apply" button which is used to restart DHCPD daemon to take into account of last saved changes (disabled),
  • the "Import" button which is used to import a dhcpd.conf file from local file-system,
  • the "Export" button which is used to export a dhcpd.conf file to local file-system.
